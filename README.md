# Jackpot Machine (Java)

A simple console-based slot machine game written in Java. The player spins the reels and wins if the symbols match.

### Features
- Random symbol generation
- Win/Lose logic
- User interaction through console input

### Concepts Practiced
- Loops and conditionals
- Random number generation
- Basic Java classes and methods
- Console input/output


## Sample Output

```
*********************
Good Luck!
|⭐|🍍|⭐|

```
